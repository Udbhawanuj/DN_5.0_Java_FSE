# Exercise 2 - Factory Method Pattern
In this exercise I created a simple document management system using the Factory Method Pattern the program creates different types of documents like Word , PDF , and Excel without directly creating their objects in the main class


## Concepts Used
* Factory Method Pattern
* Interface
* Inheritance
* Polymorphism
* Object creation through factory classes


## How to Run
```bash
javac *.java
java FactoryMethodTest
```


## Output
```text
Opening Word document...
Opening PDF document...
Opening Excel document...
```


## What I Learned
This exercise helped me understand how Factory Method keeps object creation separate from the main code it makes the program easier to manage and extend when new document types are added
