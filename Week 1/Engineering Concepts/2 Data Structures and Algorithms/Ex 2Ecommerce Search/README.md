# Exercise 2 - E-commerce Search
In this exercise I created a simple product search system in Java the program searches products by product ID using both linear search and binary search


## Concepts Used
* Array of objects
* Linear search
* Binary search
* Sorting by product ID
* Basic time complexity


## How to Run
```bash
javac *.java
java EcommerceSearchTest
```


## What I Learned
Linear search checks products one by one and works even if the data is not sorted binary search works on sorted data and is faster because it reduces the search space by half each time this exercise helped me understand why choosing the right search technique matters when the data size increases
