# Exercise 7 - Financial Forecasting
In this exercise I created a simple financial forecasting program in Java it predicts the future value based on the present value , growth rate , and number of years


## Concepts Used
* Recursion
* Formula-based calculation
* Time complexity
* Basic forecasting logic


## How to Run
```bash
javac *.java
java ForecastTest
```


## Output
```text
Present Value: 10000.0
Growth Rate: 10.0%
Years: 5
Predicted Future Value: 16105.10
```


## What I Learned
This exercise helped me understand how recursion can be used to solve repeated calculations step by step Here , each recursive call calculates the value for one year until the final future value is reached
