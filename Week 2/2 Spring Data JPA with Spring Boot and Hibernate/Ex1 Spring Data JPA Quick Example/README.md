# Ex1 - Spring Data JPA Quick Example
In this exercise I created a small Spring Boot project using Spring Data JPA and an H2 in memory database the project has a `Country` entity , a repository interface , and a service class the application saves a few country records and then reads them using Spring Data JPA methods


## Concepts Used
- Spring Boot
- Spring Data JPA
- Hibernate
- Entity class
- Repository layer
- Service layer
- H2 database
